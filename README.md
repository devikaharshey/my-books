# ✨ Words of Devika Harshey

A literary portfolio website showcasing the published and personal works of **Devika Harshey**.

This platform serves as a curated collection of 📖 poetry, 📝 creative writings, and 📚 anthology contributions, presented through an immersive editorial-inspired experience.

---

## 🌸 Features

- 📖 Dedicated showcase for personal literary works
- 📚 Publisher-wise anthology organization
- 🎨 Elegant editorial-inspired design
- ✨ Smooth scrolling and immersive animations
- 📱 Fully responsive across devices
- 🖋️ Custom typography and storytelling-focused layouts
- 💌 Contact section for readers, publishers, and collaborators

---

## 🛠️ Built With

- ⚛️ React
- 🔷 TypeScript
- 🧭 TanStack Router
- 🎨 Tailwind CSS
- 🎬 GSAP
- 🎭 Framer Motion
- ⚡ Vite

---

## 📑 Sections

### 🌟 Hero

An introduction to the author's literary journey and creative identity.

### 📖 Personal Works

A curated collection of original books, poetry collections, and independent publications.

### 📚 Anthologies

Published contributions organized by publication and publisher.

### 💌 Contact

A letter-inspired contact form for readers, publishers, and collaborators.

---

## 🎨 Design Philosophy

The website blends the aesthetics of printed literature with modern web interactions. Inspired by notebooks, manuscripts, editorial layouts, and timeless typography, it creates a calm and immersive reading experience for visitors.
